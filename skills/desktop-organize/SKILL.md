# 技能名: 桌面文件整理

## 目标
一键扫描指定目录，按文件类型自动分类并移动到对应文件夹，支持一键回滚。

## 前置工具
- `scan_files`
- `organize_directory`
- `rollback_operation`

## 执行步骤
1. 调用 `scan_files` 扫描目标目录，获取文件列表和分类统计
2. 如果文件数少于 3 个，告知用户"目录已经很整洁了"，结束
3. 调用 `organize_directory` 预览模式（dry_run=True），展示分类方案
4. 确认后调用 `organize_directory` 执行实际整理
5. 向用户报告整理结果：各分类文件数量、不确定文件列表
6. 提示用户"说「恢复」可一键撤销"

## 陷阱与检查点
- 不要整理系统目录（如 /usr, /etc）或正在使用的目录
- 文件名冲突时默认"两个都保留（重命名）"，不要擅自覆盖
- 如果目录下有 .git 等版本控制目录，跳过不处理
- 大文件（>100MB）移动前要确认
