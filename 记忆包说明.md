## 启动方式

```batch
:: 双击 start.bat，选择模式
:: 1 = Web 界面（浏览器打开 http://localhost:8080）
:: 2 = CLI 命令行模式
:: 0 = 退出
```

`start.bat` 自动完成：检测 Python → 创建 venv → 清华镜像安装依赖 → 创建 .env → 选择模式启动。
